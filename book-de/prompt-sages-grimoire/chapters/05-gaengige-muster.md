# Kapitel V: Das Muster-Kompendium

> *"Ein Muster ist ein Problem, das so oft gelöst wurde, dass die Lösung eine Form geworden ist."*

---

## Die Essentiellen Muster

Diese Muster erscheinen über Domänen hinweg. Beherrsche sie und du beherrschst den Kern des Promptings.

---

### Muster 1: Die Experten-Persona

**Zweck**: Spezifische Expertise und Kommunikationsstil hervorrufen.

**Struktur**:
```
Du bist ein [spezifischer Experten-Typ] mit [Jahren] Erfahrung in [Domäne].
Deine Spezialisierung ist [enger Fokus].
Du kommunizierst wie [Kommunikationsstil].
[Optional: spezifisches Wissen oder Perspektive einzuschließen/auszuschließen]
```

**Beispiel**:
```
Du bist ein leitender Data Scientist mit 15 Jahren Erfahrung im Finanzsektor.
Deine Spezialisierung ist die Erkennung von Betrugsmustern in Transaktionsdaten.
Du kommunizierst in klaren, jargon-freien Begriffen, weil du oft Führungskräften präsentierst.
Nimm an, ich habe statistisches Grundwissen aber keinen Machine-Learning-Hintergrund.
```

**Anti-Muster**:
```
Du bist ein Experte. (Zu vage – in was? wie expert? welcher Stil?)
```

---

### Muster 2: Die Strukturierte Ausgabe

**Zweck**: Konsistent formatierte, einfach nutzbare Ergebnisse erhalten.

**Struktur**:
```
Liefere deine Antwort in folgendem Format:

[SEKTION 1 NAME]
[Was hierhin gehört]

[SEKTION 2 NAME]
[Was hierhin gehört]

[SEKTION 3 NAME]
[Was hierhin gehört]
```

**Beispiel**:
```
Analysiere diese Geschäftsidee. Strukturiere deine Antwort als:

## GELEGENHEIT
[Welches Marktproblem löst dies?]

## DIFFERENZIERUNG
[Wie unterscheidet sich dies von existierenden Lösungen?]

## RISIKEN
[Top 3 Risiken, jedes mit Minderungs-Strategien]

## URTEIL
[Ein Absatz Bewertung mit einer Punktzahl von 1-10]
```

**Anti-Muster**:
```
Sag mir, was du denkst. (Keine Struktur = inkonsistente, schwer nutzbare Ausgabe)
```
