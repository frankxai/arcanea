# Arcanean File System

Definitions for Arcanea-specific formats and metadata conventions.

## Formats
- `.arcanea` – Realm bundle (profile data, Essence index, Spark/Shadow timeline).
- `.arc` – Essence package (prompt, output, Remix lineage, model metadata).
- `.nea` – Governance or economic record (ARC/NEA ledger entry, vote, distribution).
