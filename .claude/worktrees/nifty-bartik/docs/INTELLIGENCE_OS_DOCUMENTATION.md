## Integration Guide

### API Integration

```typescript
import { Arcanea } from '@arcanea/core';

const arcanea = new Arcanea({
  apiKey: process.env.ARCANEA_API_KEY
});

// Get Guardian guidance
const guidance = await arcanea.reason({
  query: "How to overcome creative blocks?",
  guardian: "Draconia"
});

// Activate a skill
await arcanea.activateSkill({
  skill: "dragon-breath",
  context: "career-transition"
});

// Track evolution
await arcanea.trackProgress({
  userId: "user123",
  action: "completed-project",
  xpEarned: 50
});
```

### CLI Integration

```bash
# Get Guardian guidance
arcana reason "How to build a creative practice?" --guardian Lyssandria

# Activate skills
arcana skill activate "foundation-stone" --context "new-project"

# Track evolution
arcana track progress --gate 2 --xp 150

# Access agents
arcana agent invoke "story-weaver" --topic "mythology"
```
