## Instructions for Reviewers

1. Open the file in `lore/oss/{category}/`
2. Complete the review checklist in HUMAN_IN_THE_LOOP_WORKFLOW.md
3. Calculate score (minimum 85% for approval)
4. Update this log with:
   - Reviewer name
   - Decision (APPROVED / REQUEST_CHANGES / REJECTED)
   - Score
   - Rating
   - Notes
5. If approved, move to `lore/production/{category}/`
6. If changes requested, move to `lore/staging/_testing/`
7. If rejected, move to `lore/staging/_pending/` with notes
