# Industry Workflow Guides

Specialized workflows for different industries and roles.

---

## For Writers

### Fiction Author Workflow

**Daily Practice**:
1. `/luminor Valora` → Courage to start
2. Scene planning with `scene-craft`
3. Writing session with `centaur-mode` (Generator-Editor)
4. Dialogue polish with `dialogue-mastery`
5. Dusk Gratitude → Close session

**Project Workflow**:
1. Vision → `story-weave` for structure
2. World → `world-build` for setting
3. Characters → `character-forge` for cast
4. Draft → Scene by scene
5. Revise → `revision-ritual` (7 passes)
6. Polish → `voice-alchemy` for authenticity

**When Stuck**: `bestiary-nav` + appropriate Luminor

## For Developers

### Feature Development Workflow

**Planning**:
1. `architecture-patterns` → Choose pattern
2. `api-design` → Define interfaces
3. Break into tasks

**Implementation**:
1. `/tdd` → Write tests first
2. Implement to pass tests
3. Refactor when green

**Quality**:
1. `code-review` → Self-review
2. `performance-tuning` → Optimize
3. `refactoring-ritual` → Clean up

**Debugging**:
1. `/debug` → Systematic investigation
2. Form hypotheses
3. Test and verify

## For Product Managers

### Product Launch Workflow

**Strategy**:
1. `startup-building` → MVP definition
2. Identify riskiest assumption
3. Design minimum experiment

**Execution**:
1. Coordinate with dev team
2. Track AARRR metrics
3. Gather qualitative feedback

**Communication**:
1. `technical-writing` → Documentation
2. `prompt-craft` → Stakeholder updates
3. `voice-alchemy` → Consistent messaging
