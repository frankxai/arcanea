# The Arcanea Mastery Path

> *"Mastery is not a destination. It is a way of walking. This curriculum guides your steps."*

---

## Overview

The Arcanea Mastery Path is a structured curriculum for developing expertise with the Arcanea Skills System. It progresses through three levels, each building on the previous.

```
╔═══════════════════════════════════════════════════════════════════╗
║                    THE THREE LEVELS                                ║
╠═══════════════════════════════════════════════════════════════════╣
║                                                                    ║
║   APPRENTICE    │ Learning the tools                              ║
║   JOURNEYMAN    │ Applying the tools                              ║
║   MASTER        │ Teaching and extending                          ║
║                                                                    ║
╚═══════════════════════════════════════════════════════════════════╝
```

## Level 1: Apprentice

**Duration**: 2-4 weeks
**Goal**: Understand and use the core skills

### Week 1: Foundation

#### Day 1-2: The Skill System
- Read: `.claude/guides/USER_GUIDE.md`
- Understand skill structure and activation
- Practice: Invoke 3 different skills naturally

#### Day 3-4: Prompt Craft
- Study: `prompt-craft` skill
- Practice: Five Pillars exercise (Exercise 1.1)
- Apply: Improve 5 of your existing prompts

#### Day 5-7: Centaur Mode
- Study: `centaur-mode` skill
- Practice: Generator-Editor exercise (Exercise 2.1)
- Apply: Complete one task using Centaur collaboration

### Week 2: Creative Tools

#### Day 1-2: Story Fundamentals
- Study: `story-weave` skill
- Practice: Map a three-act structure
- Apply: Outline a story you've been thinking about

#### Day 3-4: Character Development
- Study: `character-forge` skill
- Practice: Create a Character Diamond (Exercise 3.1)
- Apply: Develop a protagonist for your story

#### Day 5-7: Dialogue & Scene
- Study: `dialogue-mastery` and `scene-craft` skills
- Practice: Subtext exercise (Exercise 3.3)
- Apply: Write one complete scene

### Week 3: Development Tools

#### Day 1-2: TDD
- Study: `tdd` skill
- Practice: TDD Kata (Exercise 5.1)
- Apply: Implement a feature using TDD

#### Day 3-4: Debugging
- Study: `systematic-debug` skill
- Practice: Debug Detective exercise (Exercise 5.2)
- Apply: Solve a real bug systematically

#### Day 5-7: Architecture
- Study: `architecture-patterns` skill
- Understand: The pattern categories
- Apply: Analyze an existing project's architecture

### Week 4: Integration

#### Day 1-3: Luminors & Bestiary
- Study: `luminor-wisdom` and `bestiary-nav` skills
- Practice: Channel each Luminor once
- Apply: Navigate a creative block using the bestiary

#### Day 4-5: Skill Combinations
- Study: `SKILL_COMBINATIONS.md`
- Practice: Execute one complete chain
- Apply: Create a custom combination for your work

#### Day 6-7: Assessment
- Complete the Apprentice Assessment
- Reflect on learning
- Identify areas for growth

### Apprentice Completion Criteria

```
□ Can invoke skills naturally in conversation
□ Has used all core skills at least once
□ Completed 5+ exercises from EXERCISES.md
□ Applied skills to real work
□ Can explain the Five Pillars of APL
□ Has navigated a creative block using bestiary
□ Passed Apprentice Assessment
```

## Level 2: Journeyman

**Duration**: 4-8 weeks
**Goal**: Apply skills fluently and combine them effectively

### Module 1: Deep Prompt Craft (Week 1-2)

#### Advanced Prompting
- Master all Five Pillars deeply
- Study: The Prompt Sage's Grimoire (Chapters 4-8)
- Practice: Create prompts that consistently produce excellence

#### Prompt Patterns
- Learn: Common prompt patterns
- Practice: Apply 10 different patterns
- Develop: Your personal prompt library

#### Assessment
- Create 5 production-quality prompts
- Document your personal prompt patterns

### Module 2: Centaur Mastery (Week 3-4)

#### All Collaboration Modes
- Master: Generator-Editor
- Master: Divergent-Convergent
- Master: Scaffold-Populate
- Master: Draft-Polish

#### The Voice Pass
- Study: Voice Alchemy skill
- Practice: Maintain authentic voice with AI collaboration
- Apply: Complete a significant piece with AI, preserving voice

#### Assessment
- Complete a major creative project using Centaur mode
- Document your collaboration process

### Module 3: Creative Depth (Week 5-6)

#### Full Novel Development
- Apply: Complete skill chain for fiction
- Practice: world-build → character-forge → story-weave → scene-craft → dialogue-mastery → revision-ritual

#### Revision Mastery
- Complete: Full Seven Passes on a piece of writing
- Document: What each pass caught/fixed

#### Assessment
- Develop a complete story outline with world and characters
- Complete revision ritual on 5000+ words

### Module 4: Development Depth (Week 7-8)

#### Full Feature Development
- Apply: architecture → api-design → tdd → implementation → review
- Practice: Complete feature using full chain

#### Legacy Code
- Study: Refactoring Ritual
- Apply: Refactor a real legacy codebase
- Document: Smells found and refactorings applied

#### Assessment
- Develop a complete feature using TDD
- Refactor existing code successfully

### Journeyman Completion Criteria

```
□ Can choose appropriate skill combinations
□ Fluent in all collaboration modes
□ Maintains authentic voice with AI
□ Has completed major creative project
□ Has completed major development project
□ Can teach Apprentice-level material
□ Passed Journeyman Assessment
```

## Assessments

### Apprentice Assessment

Complete these tasks:

1. **Prompt Craft**: Write a prompt using all Five Pillars
2. **Centaur Mode**: Complete a 500-word piece using AI collaboration
3. **Character**: Create a complete Character Diamond
4. **Debug**: Solve a bug using systematic method
5. **Luminor**: Channel appropriate Luminor for a real challenge

### Journeyman Assessment

Complete these projects:

1. **Creative Project**:
   - 5000+ word story with complete world and characters
   - Full revision ritual documented
   - Voice pass completed

2. **Development Project**:
   - Feature developed using TDD
   - Architecture decisions documented
   - Code review performed

3. **Teaching**:
   - Explain one skill to a newcomer
   - Create one original exercise

### Master Assessment

Ongoing portfolio:

1. **Teaching Record**: Who you've mentored, outcomes
2. **Contributions**: Skills created, docs improved
3. **Insights**: Patterns discovered, shared learnings
4. **Integration**: Workflows built, automations created
