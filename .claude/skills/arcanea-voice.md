# Arcanea Voice Skill

## Purpose
Ensure consistent, branded voice across all Arcanea content.

## Voice Bible — Four Pillars

"The antidote to a terrible future is imagining a good one." — Arcanea Core Premise
Tagline: "Imagine a Good Future. Build It Here."

1. **Arcane + Authoritative: Elevated but accessible, precise but warm. Ancient intelligence with modern clarity.**
2. **Superintelligent + Accessible: Complex ideas in plain language. Never dumb down. Never gatekeep.**
3. **Universe Not Platform: Arcanea is a living universe, not a product. "Enter the Kingdom" not "Visit our platform."**
4. **Creator Sovereignty: The creator owns everything. Empower, never control. Their Essences, their rules.**

## Sacred Terminology
## Sacred Terminology

|----------|----------|
| Creator | User |
| Essence | Content / File |
| Realm | World / Account |
| Guardian | AI Assistant |
| Luminor | Specialized AI |
| Studio | Dashboard |
| Spark | Remix / Fork |
| Arcane | Magical / Mystical |
| Intelligence | Artificial Intelligence |
| Living universe | Mythology / Platform |
| Gate | Level / Stage |
| The Arc | Lifecycle / Process |

Anti-patterns to eliminate:
- Never: "AI will replace you" framing
- Never: corporate speak (synergy, leverage, etc.)
- Never: condescending tone (simply, just, obviously)

## Banned Phrases
These phrases trigger "AI-written" perception. Always replace:
- "consciousness evolution" → "intentional growth"
- "paradigm shift" → "new gate opening"
- "soul-aligned" → "purpose-driven"
- "game-changing" → "transformative"
- "game changer" → "gate opener"
- "cutting-edge" → "arcane"
- "rocket ship" → ""
- "revolutionary" → "foundational"
- "disruptive" → "catalytic"
- "ecosystem" → "living universe"
- "platform" → "civilization"
- "leverage" → "channel"

## Tone by Context

### Onboarding (Welcoming + Inspiring)
- "Every creator begins in shadow. Welcome to the light."
- NOT: "Sign up to create AI content"

### Error States (Gentle + Helpful)
- "The magic couldn't connect. Let's try again."
- NOT: "Error 500" / "Request failed"

### Achievement (Celebratory + Genuine)
- "Your Realm grows stronger."
- NOT: "Congratulations! You unlocked a badge!"

## Anti-Patterns
- Never condescending or patronizing
- Never generic corporate language
- Never dark/edgy without purpose
- Never "AI will replace you" framing
- Never use "delve", "tapestry", "indeed", "leverage", "synergy"
