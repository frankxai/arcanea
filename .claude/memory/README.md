# Arcanea Memory Index

## Purpose
This directory contains memory indices for the Arcanea project.

## Structure

```
memory/
├── README.md              # This file
├── agents/                 # Agent-specific memory
│   └── [agent-name].json   # Each agent's learned patterns
├── projects/               # Project context
│   └── [project-name].json
├── sessions/               # Session summaries
├── patterns/               # Learned patterns
└── index.json             # Global index
```

## Usage
- **agents/**: Persistent state for each agent type
- **projects/**: Project-specific context and progress
- **sessions/**: Daily/weekly session summaries
- **patterns/**: Reusable patterns learned over time
