## Contribution Guidelines

### Voice and Tone

Arcanea has a distinctive voice:

- **Elevated but accessible** - Wisdom, not pretension
- **Mythic but practical** - Always actionable
- **Universal** - No cultural appropriation
- **Direct** - No filler or fluff

### Quality Standards

Before submitting:

1. **Test it yourself** - Use your contribution in real work
2. **Follow the template** - Consistency matters
3. **Be specific** - Generic advice doesn't help
4. **Be original** - Don't duplicate existing content

### Process

1. Fork the repository
2. Create your contribution following the relevant template
3. Test thoroughly
4. Submit a pull request with:
   - Clear description of what you're adding
   - Why it's needed
   - How you've tested it
