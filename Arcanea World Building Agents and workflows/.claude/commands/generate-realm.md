## Step 5: Agent Orchestration

I will coordinate ALL agents to build your world:

### Phase 1: Foundation (Lore Master + All Departments)
1. Create `world.arcanea` configuration
2. Generate `foundations/cosmology.md`
3. Generate `foundations/natural-laws.md`
4. Generate `foundations/history-timeline.md`
5. Generate `foundations/magic-system.md` (if applicable)

### Phase 2: Geography (World Architect + Specialists)
1. Design primary realm/continent
2. Create major regions
3. Establish key locations
4. Map climate and resources

### Phase 3: Cultures (World Architect + Culture Anthropologist)
1. Design primary cultures
2. Establish customs and governance
3. Define cultural conflicts

### Phase 4: Characters (Character Weaver)
1. Create protagonist(s)
2. Create antagonist(s)
3. Create supporting cast
4. Map relationships

### Phase 5: Magic (Magic Systems - if applicable)
1. Detail magic categories
2. Create signature artifacts
3. Define magical phenomena

### Phase 6: Narrative (Narrative Director)
1. Design major conflicts
2. Create quest hooks
3. Establish prophecies/legends

### Phase 7: Validation (Lore Master + Consistency Validator)
1. Run full consistency check
2. Validate all relationships
3. Approve for canon

The complete world will be created in:
`worlds/[world-name]/`
