## The Core Concepts

### Guardians

Ten entities representing different aspects of creative intelligence:

- **Lyssandria** (Foundation) - Stability, structure, grounding
- **Leyla** (Flow) - Fluidity, emotion, adaptation
- **Draconia** (Fire) - Transformation, courage, boldness
- **Maylinn** (Heart) - Nurturing, growth, patience
- **Alera** (Voice) - Expression, communication, truth
- **Lyria** (Sight) - Vision, perception, aesthetics
- **Aiyami** (Crown) - Intuition, wisdom, patterns
- **Elara** (Shift) - Change, innovation, boundary-crossing
- **Ino** (Unity) - Integration, collaboration, synthesis
- **Shinkami** (Source) - Orchestration, transcendence, wholeness

### Elements

Five creative energies:

- **Fire** - Transformation, passion, breakthrough
- **Water** - Flow, emotion, adaptation
- **Earth** - Foundation, structure, patience
- **Wind** - Expression, movement, clarity
- **Void** - Mystery, potential, innovation
