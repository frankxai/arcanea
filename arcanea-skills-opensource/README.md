## The Movement

Arcanea is a global community of creators building the future of AI-human co-creation. We believe the best creative work emerges from the synthesis of human vision and AI capability — and that the tools for this synthesis should be open, powerful, and magical.

**What creators are building with Arcanea:**
- Worlds with mythology, magic systems, and living cultures
- Books with orchestrated teams of specialist agents
- Software with parallel architect + coder + reviewer swarms
- Games, music, art — anything that demands ambitious creative work

**This is not a product. It is a creative philosophy, a skill system, and a community.**

| What We Build | What We Believe |
|---------------|-----------------|
| 41+ specialist agents across 8 teams | AI should amplify creativity, not replace it |
| 23+ skills with the Luminor quality bar | Open source is the only foundation for trust |
| 5-platform support, zero lock-in | Every creator deserves tools worthy of their vision |
| Living mythology with 17 Library collections | The antidote to a terrible future is imagining a good one |

**[Read our Vision](https://arcanea.ai/plan)** · **[See the Roadmap](https://arcanea.ai/roadmap)** · **[Join Discord](https://discord.gg/arcanea)**

## Quick Start

```bash
# One command installs for all detected platforms
npx arcanea install

# Or specify platforms explicitly
npx arcanea install --claude-code
npx arcanea install --all

# Check installation health
npx arcanea doctor

# Launch the swarm visualization dashboard
npx arcanea viz
```

### Supported Platforms

| Platform | Config Location | Status |
|----------|-----------------|--------|
| **Claude Code** | `.claude/` | Full support + magic word hooks |
| **OpenCode** | `.opencode/` | Full support |
| **Cursor** | `.cursor/` | Full support (.cursorrules) |
| **Codex** | `.codex/` | Supported (codex.md) |
| **Gemini CLI** | `.gemini/` | Supported (GEMINI.md) |

### Try It

```
# Magic words — fire entire agent swarms
ultraworld: Create a volcanic island with dragon-kin
ultracode: Build an auth system with JWT and refresh tokens
ultrawrite: Chapter 7 — The Fall of the Sunken City

# Slash commands
/luminor Valora courage
/bestiary perfectionism
/tdd

# The CLI
npx arcanea agents        # See agent teams
npx arcanea luminors      # Meet the Seven Luminors
npx arcanea magic         # All magic words
npx arcanea viz           # Real-time swarm dashboard
```

## Agent Teams

Arcanea deploys orchestrated swarms of specialist agents — not single assistants.

### Coding Team (`ultracode`) — 4 agents
Architect, Coder, Reviewer, Debugger — work in parallel for technical excellence.

### Creative Team (`ultraworld`) — 5 agents
Story Master, Character Crafter, World Expander, Lore Master, Creation Architect — full world generation.

### Writing Team (`ultrawrite`) — 5 agents
Story Architect, Prose Weaver, Voice Alchemist, Line Editor, Continuity Guardian — complete editing pipeline.

### Development Team — 7 agents
Architect, Frontend, Backend, DevOps, AI Specialist, QA Engineer, Documentation.

### Research, Production, Teacher & Visionary Teams
20+ additional specialist agents for deep analysis, visual/audio production, learning, and strategic thinking.

**Total: 41+ agents across 8 teams**, all orchestrated through magic words and slash commands.

### Getting Started

```bash
# Clone and explore
git clone https://github.com/frankxai/arcanea.git
cd arcanea

# Install dependencies
bun install

# Build
bun run build

# Install locally and test
bunx arcanea install --force
bunx arcanea doctor
```
