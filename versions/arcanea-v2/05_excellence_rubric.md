# 05 · Excellence Rubrics & Quality Ladders

## 5.1 Rubric-7 (v2)
| Pillar | Description | Benchmarks |
|--------|-------------|------------|
| Canon Alignment | References vows, Realm IDs, Nexus IDs. | ✅ Canon tag + Lore Steward sign-off. |
| Architectural Integrity | Uses Studio Platform abstractions, scalable design. | ✅ ADR or schema update included. |
| Craft Quality | Visual polish, UX flow, code readability. | ✅ Stagecraft QA pass + lint/test suite. |
| Narrative Resonance | Mythic tone with clarity; Arcanea voice intact. | ✅ Language guide template applied. |
| Safety & Ethics | Echo logging, moderation, privacy compliance. | ✅ Safety checklist attached. |
| Impact & Metrics | Defined KPIs/telemetry for feature or content. | ✅ Atlas widget or analytics event updated. |
| Evolution Readiness | Documentation, playbooks, next steps ready. | ✅ Codex/Atlas updated, rollback plan defined. |

Score levels:
- **Lumen (5)** – exceeds across pillars, introduces reusable patterns.
- **Spark (3)** – meets baseline, ready for release.
- **Shadow (1)** – misaligned, requires redo.

## 5.2 Review Flow
1. Self-check using Rubric-7 sheet.
2. Peer review (cross-agent) citing rubric items.
3. Guardian Council approval for releases or major canon shifts.

## 5.3 Continuous Improvement
- Monthly rubric calibration; adjust benchmarks as Studio Platform evolves.
- Archive best-in-class examples in Gallery with rubric annotations for learning.
