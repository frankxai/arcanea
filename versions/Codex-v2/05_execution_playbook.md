# 05 · Execution Playbook

## 5.1 Cadence Overview
- **Daily** – Pulse Sync (15 min): review Echo logs, top three priorities, Guardian support.
- **Weekly** – Sanctuary Council: strategy alignment, Studio KPIs, canon updates.
- **Lunar (28-day)** – Realm Convergence: release notes, highlight Soul Guardians/Spark stories, community showcase.
- **Quarterly** – Prime Conclave: roadmap recalibration, financial review, partner readiness.

## 5.2 OKR Framework (Spark Cycle)
| Objective | Key Results |
|-----------|-------------|
| Deliver Arcanea Studio MVP | Studio Portal live, Nexus concierge assisting prompts, Remix metadata shipped |
| Activate Arcanean Profiles | Profiles beta with Realm pages, Sparks/Shadows visible, remix button functioning |
| Establish Guardian Operations | Moderation pipeline live, Echo dashboards in Sanctuary, Rubric-7 adoption across teams |

## 5.3 Ritual Scripts
- **Pulse Sync** – use language guide template, share Spark highlight + Shadow to resolve.
- **Guardian Retrospective** – collect luminous wins, shadows learned, vow commitments.
- **Echo Audit** – weekly log review for anomalies; escalate to Sanctuary for resolution.

## 5.4 Tooling Stack
- Project tracking: Linear (tags `studio/*`, `profiles/*`, `guardian/*`), Notion ritual calendar.
- Analytics: Mixpanel (Studio/Profile usage), Supabase dashboards (Echo metrics), Grafana (model telemetry).
- Communication: Slack (`#arcanea-studio`, `#guardian-council`, `#soul-guardians`), Discord community hub.

## 5.5 Release Playbook
1. Draft release plan referencing vow IDs + impact areas.
2. Update Codex/Atlas copy and Studio strings.
3. Run design QA (Stagecraft checklist) + automated/manual testing + moderation gating.
4. Deploy via Turborepo pipeline.
5. Announce with premium Arcanea voice, update Echo logs, spotlight in Gallery timeline.

## 5.6 Role Matrix
| Role | Responsibilities |
|------|------------------|
| Lore Steward | Maintains canon, approves language shifts. |
| Product Architect | Aligns architecture with Studio Platform vision, signs ADRs. |
| Guardian Council | Oversees rituals, quality, safety. |
| Echo Keeper | Moderation, logging, compliance reporting. |
| Stagewright (Engineering Lead) | Delivers Studio features, enforces vows. |
| Maestro (Design Lead) | Upholds Stagecraft Palette and experience polish. |

## 5.7 Metrics Dashboard
- **Canon Integrity** – % of assets referencing latest vows and terminology.
- **Studio Engagement** – Essences generated, remix rate, model success rate, cost per Essence.
- **Profile Growth** – Active Creators, Realm followers, remix-to-share conversion.
- **Guardian Operations** – moderation latency, Rubric-7 compliance, Echo incident count.

## 5.8 Adaptation Protocol
- New Realms/Luminors require updates to Foundational Truths, Language Guide, Studio copy.
- Run mini-conclave for approvals; archive superseded docs in `docs/legacy/` with context.
- Review roadmap quarterly to align with Spark → Field → Realm phases.
