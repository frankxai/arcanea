Component Locking System

Three-tier system to preserve perfect components while allowing customization where needed.

🔒

LOCKED Components

Perfected components that should NOT be modified. These represent the premium Arcanea aesthetic:

  • • Premium buttons with light sweeps
  • • Glass buttons with shimmer effects
  • • Lift and magnetic hover effects
  • • All grained gradients
  • • Character-specific theming
🔧

FLEXIBLE Components

Customizable components you can modify and extend as needed:

  • • Basic card structures
  • • Simple button variants
  • • Typography foundations
  • • Layout grids and containers
  • • Form elements and inputs
⚠️

EXPERIMENTAL

Work-in-progress components being tested and refined:

  • • Advanced glass effects
  • • Complex hover animations
  • • New gradient techniques
  • • Experimental interactions
  • • 3D transform effects

Usage Examples 

/* 🔒 LOCKED - Use as-is, don't modify */
.btn-premium { /* Perfect premium button */ }
.hover-lift-premium { /* Perfect lift effect */ }
.bg-grained-aurora { /* Perfect grained gradient */ }

/* 🔧 FLEXIBLE - Customize freely */  
.card-flexible { /* Modify colors, sizes, etc. */ }
.btn-flexible-primary { /* Create variants */ }
.grid-flexible { /* Adjust layout as needed */ }

/* ⚠️ EXPERIMENTAL - Use with caution */
.glass-experimental { /* May change */ }
.hover-experimental { /* Being refined */ }

Why This System?

🎯 Preserve Excellence

The perfect button hover effects, magical interactions, and grained gradients are locked to maintain the premium Arcanea aesthetic. These took significant effort to perfect and should be preserved.

🚀 Enable Innovation

Flexible components provide building blocks for custom designs while experimental components let us test new ideas without affecting production-ready elements.
