🚀 Utility-First Philosophy

This implementation follows Tailwind CSS's utility-first approach, combining atomic CSS classes with Arcanea's premium design tokens and effects. Build any design directly in your markup with consistent, predictable utilities.

📚 Usage Guide

🎯 Quick Start

<!-- Basic button -->
<button class="btn-tailwind btn-primary">
  Click me
</button>

<!-- Glass card -->
<div class="card-glass">
  <h3 class="font-semibold mb-2">Title</h3>
  <p class="text-gray-600">Content</p>
</div>

⚡ Custom Combinations

<!-- Custom hover effects -->
<div class="p-6 glass rounded-xl 
     hover:lift-premium 
     hover:shadow-premium 
     transition-premium">
  Custom component
</div>

💡 Pro Tips

  • • Combine utilities for unique designs: glass hover:lift-premium transition-premium
  • • Use responsive prefixes: sm:grid-cols-2 md:grid-cols-3
  • • Layer premium effects: bg-gradient-aurora hover:shadow-premium
  • • Focus states included: focus:ring-2 focus:outline-none

⚛️ React + Headless UI

import { Dialog } from '@headlessui/react'

function PremiumModal({ isOpen, onClose }) {
  return (
    <Dialog 
      open={isOpen} 
      onClose={onClose}
      className="fixed inset-0 z-50 flex 
                 items-center justify-center p-4"
    >
      <Dialog.Overlay className="fixed inset-0 
                                bg-black bg-opacity-25" />
      
      <div className="card-glass max-w-md mx-auto">
        <Dialog.Title className="text-lg font-bold 
                                mb-4 text-primary">
          Premium Modal
        </Dialog.Title>
        
        <p className="text-gray-600 mb-4">
          Modal content with glass morphism.
        </p>
        
        <button 
          onClick={onClose}
          className="btn-tailwind btn-primary w-full"
        >
          Close
        </button>
      </div>
    </Dialog>
  )
}
