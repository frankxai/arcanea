# Arcanea Community Protocols

> *"No creator walks alone through the Gates."*

## Community Architecture

### Digital Spaces

#### Discord Server — The Academy
- **#entrance-hall** — Welcome, onboarding, Gate quiz
- **#foundation-gate** — Infrastructure, databases, systems (Lyssandria)
- **#flow-gate** — Art, music, design, creativity (Leyla)
- **#fire-gate** — Code challenges, performance, action (Draconia)
- **#heart-gate** — Support, mentoring, collaboration (Maylinn)
- **#voice-gate** — Writing, content, storytelling (Alera)
- **#sight-gate** — Research, analysis, insights (Lyria)
- **#crown-gate** — Philosophy, learning, wisdom (Aiyami)
- **#shift-gate** — Change management, migrations (Elara)
- **#unity-gate** — Cross-discipline projects (Ino)
- **#source-gate** — Meta-discussions, architecture (Shinkami)
- **#the-library** — Book club, text discussions
- **#bestiary** — Share creative blocks and victories
- **#luminor-council** — Advanced creators (rank-gated)

#### GitHub — The Forge
- Open-source packages for community contribution
- Issue templates aligned with Guardian domains
- PR templates with Gate alignment
- Community challenges as GitHub discussions

#### Arcanea Platform (arcanea.ai)
- Creator profiles with Gate progress
- AI companion customization
- Content sharing and curation
- Community challenges and quests

### Physical Spaces

#### Monthly Gatherings — Gate Ceremonies
- **Format**: 2-hour themed creative sessions
- **Structure**:
  1. Opening invocation (10 min)
  2. Guardian teaching (20 min)
  3. Creative workshop (60 min)
  4. Sharing circle (20 min)
  5. Closing ritual (10 min)
- **Themes rotate**: One Gate per month (10-month cycle)

#### Quarterly Summits — The Convergence
- Full-day events combining all Gates
- Keynotes, workshops, demonstrations
- Community awards and rank ceremonies
- Evening creative showcase

#### Annual Festival — The Luminor Gathering
- Multi-day immersive experience
- Live music, art installations, workshops
- The Great Creation — collaborative artwork
- Luminor ceremony for those who opened all Gates

## Engagement Systems

### Quest System
- **Daily Quests**: Small creative tasks (5-15 min)
- **Weekly Challenges**: Gate-specific deeper work
- **Monthly Missions**: Cross-Gate collaborative projects
- **Seasonal Arcs**: Multi-month narrative campaigns

### Rank Progression
Members progress through ranks by completing quests and opening Gates:
- **Apprentice** (0-2 Gates) — Learning fundamentals
- **Mage** (3-4 Gates) — Active creation
- **Master** (5-6 Gates) — Teaching others
- **Archmage** (7-8 Gates) — Leading projects
- **Luminor** (9-10 Gates) — Guiding the community

### Recognition
- **Guardian's Choice** — Weekly spotlight by rotating Guardian
- **The Luminor Board** — Top contributors displayed on platform
- **Gate Mastery Badges** — NFT-backed achievement tokens
- **The Chronicle** — Monthly newsletter featuring community stories

## Moderation

### The Guardian Code
1. **Respect the Creator** — Every person is on their own Gate journey
2. **Honor the Work** — Constructive feedback only, never destructive
3. **Protect the Space** — Report toxicity, defend the vulnerable
4. **Share the Light** — Knowledge shared is knowledge doubled
5. **Walk the Path** — Lead by example, not decree

### Enforcement Tiers
1. **Gentle Reminder** — Private message, Guardian voice
2. **Cooling Period** — 24-hour timeout, reflection prompt
3. **Gate Closure** — 7-day restricted access
4. **Banishment** — Permanent removal (reserved for severe violations)
