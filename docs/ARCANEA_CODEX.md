  * 20% Community sale/airdrop
  * 10% Partners
  * 10% Liquidity
  * 10% Creator rewards

### Utilities

* API credits (20% cheaper in ARC).
* Minting worlds, LoRAs, assets.
* Unlocking Academy lessons.

### NFTs
