# Arcanea Canvas - Product Vision

> "Spatial Intelligence for Creators"

## What Is Arcanea Canvas?

A browser-based spatial operating system where humans and AI agents co-create. Not a drawing app - a thinking surface where ideas become structures, structures become plans, and plans become execution.

## Core Capabilities

### Spatial Thinking
- Infinite canvas with zoom/pan
- Place any content anywhere - notes, documents, code, images, live previews
- Connect ideas visually with arrows and grouping frames
- Spatial arrangement conveys meaning

### AI-Native Intelligence
- AI agents generate content directly on the board
- Select objects → ask AI to summarize, expand, connect, transform
- Drop a URL → auto-extract content (YouTube transcripts, article text, metadata)
- Generate multiple forms: text, mindmaps, images, infographics, UI mockups, code

### Live Content
- Embed live web previews (localhost, GitHub Pages, any URL)
- Render AI-generated HTML/UI directly in sandboxed iframes
- Code blocks with syntax highlighting
- Markdown documents with full rendering

### Semantic Foundation
- Every visual object has semantic metadata (title, tags, relations, provenance)
- AI agents operate on the semantic model, not just pixels
- Queryable graph of all board objects and relationships
- Full version history

## Target Users

1. **Creators** building worlds, stories, and content systems
2. **Developers** planning architectures and workflows
3. **Teams** mapping knowledge and coordinating projects
4. **AI practitioners** designing agent workflows and pipelines

## Competitive Position

| Arcanea Canvas | vs. Miro | vs. Excalidraw | vs. Figma |
|----------------|----------|----------------|-----------|
| AI-native from core | AI add-on | No AI | AI add-on |
| Rich content nodes | Sticky notes + shapes | Shapes only | Design-focused |
| Live previews | Limited embeds | None | Prototyping only |
| Semantic model | None | None | None |
| Agent-operable | Limited | No | No |
| Open architecture | SaaS lock-in | MIT but limited | SaaS lock-in |

## Product Principles

1. **Intelligence over interface** - The canvas understands what's on it
2. **Generation over decoration** - AI creates content, not just arranges it
3. **Spatial is semantic** - Position and proximity carry meaning
4. **Open by design** - Agents, APIs, and exports; no lock-in
5. **Beauty as default** - Arcanea cosmic design system makes everything premium
