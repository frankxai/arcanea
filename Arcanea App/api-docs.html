POST /api/chat

Send a message to the Arcanea SuperAgent and receive an AI-generated response. Supports streaming responses and conversation context.

Request Parameters

Parameter Type Required Description
messages Array Yes Array of message objects with role and content
model String No AI model to use (default: gpt-4-turbo-preview)
temperature Number No Creativity level 0-1 (default: 0.7)
stream Boolean No Enable streaming responses (default: true)
Request Example
{
  "messages": [
    {
      "role": "user",
      "content": "Help me write a fantasy novel opening"
    }
  ],
  "model": "gpt-4-turbo-preview",
  "temperature": 0.8,
  "stream": true
}

Response

Success 200
Error 400
{
  "id": "msg_abc123",
  "object": "chat.completion",
  "created": 1699999999,
  "model": "gpt-4-turbo-preview",
  "choices": [{
    "message": {
      "role": "assistant",
      "content": "The mist rolled across the ancient bridge like..."
    },
    "finish_reason": "stop"
  }]
}
{
  "error": {
    "message": "Invalid request format",
    "type": "invalid_request_error",
    "code": "invalid_messages"
  }
}
⚠️ Rate Limits

60 requests per minute per user

POST /api/luminors/scripta

Access Scripta, the book authoring Luminor for writing assistance and content generation.

Request Parameters

Parameter Type Required Description
action String Yes Action type: generate, edit, analyze, export
content String No Existing content to work with
prompt String Yes Instructions for the Luminor
format String No Output format: text, markdown, pdf, epub
POST /api/luminors/lumina

Generate images using Lumina, the AI image generation Luminor powered by DALL-E 3.

Request Parameters

Parameter Type Required Description
prompt String Yes Image description prompt
style String No Art style: photorealistic, digital-art, anime, etc.
size String No Image size: 1024x1024, 1792x1024, 1024x1792
quality String No Quality level: standard or hd
POST /api/luminors/kinetix

Create videos using Kinetix, the AI video generation Luminor.

Request Parameters

Parameter Type Required Description
prompt String Yes Video scene description
duration Number No Video duration in seconds (15, 30, 60)
style String No Video style: cinematic, animation, documentary
resolution String No Output resolution: 720p, 1080p, 4k
GET /api/content/projects

Retrieve all user projects across different Luminors.

POST /api/content/save

Save project content to the Arcanea cloud storage.

POST /api/content/export

Export content in various formats (PDF, EPUB, MP4, etc.).

POST /api/auth/login

Authenticate user and receive access tokens.

Headers

Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

API Key Management

API keys can be generated from your Arcanea dashboard. Keep your API keys secure and never expose them in client-side code. Use environment variables to store sensitive credentials.

⚠️ Security Best Practices
  • Rotate API keys regularly
  • Use separate keys for development and production
  • Implement IP whitelisting when possible
  • Monitor API usage for unusual patterns
POST /api/webhooks/configure

Configure webhook endpoints to receive real-time updates about content generation, project changes, and system events.

Available Events

  • content.generated - When new content is created
  • project.updated - When a project is modified
  • export.completed - When export process finishes
  • luminor.response - When a Luminor completes processing

Webhook Payload Example

{
  "event": "content.generated",
  "timestamp": "2024-01-15T10:30:00Z",
  "data": {
    "project_id": "proj_123",
    "luminor": "scripta",
    "content_type": "chapter",
    "word_count": 2500
  }
}
