## SEO Strategy

### Keyword Layers

**Primary (Universe):**
- Arcanea, Lumina and Nero, Ten Gates, Guardian names

**Secondary (Creator):**
- Creative process framework, overcoming blocks, systematic mastery

**Long-Tail (Problem):**
- How to overcome creative resistance, dealing with perfectionism

### Meta Description Formula

```
[Myth/Concept] + [Practical Application] + [Unique Framework] + CTA
```

**Example:**
> "Lumina and Nero's cosmic partnership reveals the secret of all creation: balance between form and potential. Learn the First Dawn principle and apply it to your work today."

### Internal Linking Model

**Hub-and-Spoke:**
- Hubs: Academy Handbook, Canon, Bestiary Intro, Rituals Index
- Spokes: Individual texts linking to 1 hub + 2-3 related + 1-2 deeper

**Target:** 5-8 links per text
- 40% same collection
- 40% cross-collection
- 20% canonical reference

## Quality Standards

### Readability

- **Target:** 9th-10th grade Flesch-Kincaid
- **Paragraphs:** Max 3-4 sentences
- **Sentences:** Average 15-20 words
- **Whitespace:** Generous line breaks, lists, blockquotes
- **Hierarchy:** Clear H2/H3 structure

### Voice

- **Personal stories** with specific details (times, numbers, results)
- **Guardian integration** natural and relevant
- **Balance** mythic elevation with practical grounding
- **Avoid** generic self-help language
- **Include** vulnerability and failures

### Canonical

- Lumina/Nero as partnership (not good vs evil)
- Ten Gates with proper names, frequencies
- Five Elements including Void/Spirit duality
- Guardian-Godbeast pairs correct
- Magic terminology consistent
- The Arc for cycles/transformation

## Polish Checklist

**Every text must pass before "ready to publish":**

Voice & Style (8 checks)
- Opens with hook
- FrankX first-person voice
- Personal example included
- Balances mythic/practical
- Avoids generic language
- Studio-rooted authenticity
- Strong closing
- Cinematic yet intimate feel

Canonical Alignment (8 checks)
- Lumina/Nero duality correct
- Guardian names/associations correct
- Gates with number/name/frequency
- Elements including Void/Spirit
- Magic terminology consistent
- The Arc referenced appropriately
- No generic light/dark
- Names match canon

Structure & Readability (8 checks)
- Clear H2/H3 hierarchy
- Short paragraphs (3-4 sentences)
- Generous whitespace
- Lists for sequences
- Blockquotes for principles
- Bold/italic purposeful
- Tables for comparisons
- Reading level 9-10 grade

SEO & Discoverability (8 checks)
- Title optimized (50-60 chars)
- Meta description (150-160 chars)
- Primary keyword identified
- URL slug optimized
- H1 includes keyword
- Subheadings include keywords
- Internal links descriptive
- External links quality

Cross-Linking (8 checks)
- 3+ internal links in body
- "See Also" section (4-7 links)
- Links to hub content
- Links to same collection
- Links to practices
- Links to diagnostics
- No broken links
- Link descriptions clear

Metadata (8 checks)
- Collection name
- Theme/element
- Gate if applicable
- Guardian if applicable
- Difficulty level
- Read time
- Practice time
- Frequency

Final Quality (8 checks)
- Spell-checked
- Grammar clean
- Formatting consistent
- No placeholders
- Quotes attributed
- Terms defined
- No orphaned references
- Tested in preview

**Total:** 56-point checklist
